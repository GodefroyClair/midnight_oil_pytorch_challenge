## Notes:

If possible, all the text files have to be formatted with the [Markdown format](https://github.com/adam-p/markdown-here/wiki/Markdown-Cheatsheet)
Examples of conversion with the command line [pandoc tool](https://pandoc.org/):

- To PDF
$pandoc reu-midnight-oil-1.txt -o reu-midnight-oil1.pdf
- To HTML:
$pandoc reu-midnight-oil-1.txt -o reu-midnight-oil1.html
